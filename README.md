# core tech 

## 1. ZK 技术（零知识证明）—— 尤其是 ZK-EVM 和 zkSync

    实现 完全隐私 + 高性能扩容
    支持以太坊兼容的智能合约在 L2 上运行
    被称为“以太坊的终极扩容方案”


## 2. 模块化区块链（Modular Blockchain）

将区块链拆分为多个层（执行、共识、数据可用性、结算）

## 3. Restaking & 主动验证服务（AVS）

原理：允许用户将其质押的 ETH “再质押”给其他协议，提供信任共享安全
